# LLM Storytelling

Проект по обучению и тонкой настройке (fine-tuning) языковых моделей для генерации структурированных детских сказок.

## Основные возможности
* Подготовка и очистка датасетов сказок (CSV -> JSONL).
* Тонкая настройка моделей (Fine-tuning).
* Оценка качества генерации и валидация результатов.

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Vitaliiskyi/llm-storytelling.git
cd llm-storytelling
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv .venv
.venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Структура проекта

* `data_prep.py` — скрипт для обработки сырых данных и подготовки их к обучению.
* `train.py` — основной скрипт для запуска процесса fine-tuning.
* `eval_model.py` — оценка обученной модели и генерация примеров.
* `config.py` — конфигурация параметров обучения и путей к данным.
* `utils.py` — вспомогательные функции.
* `playground/` — папка для экспериментов и черновых скриптов.
